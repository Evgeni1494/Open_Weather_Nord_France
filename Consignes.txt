Récupérer les informations suivantes et stocker les dans un DataFrame (pandas) pour 20 villes françaises :

    Température actuelle ok
    Température ressentie ok
    Température minimale et maximale ok
    Pression atmosphérique ok
    Humidité ok
    Vitesse du vent ok
    Direction du vent ok
    Description du temps OK
    Lever du soleil (Attention à bien convertir en information compréhensible pour un humain) ok
    Coucher du soleil (Attention à bien convertir en information compréhensible pour un humain) ok

​

Vous devez récupérer les informations actuelles. Les informations seront d'abord stockés dans un DataFrame et ensuite vous exporterez ce dataframe dans un fichier csv.

​

Vous devrez faire une veille technologique afin de monter en compétences.
